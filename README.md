## Description

Selfe service checker API with NestJs.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## DB

Run Postgress db indocker with: docker-compose up -d
Seed for Currencies: ts-node currency.seed.ts
